#Portfolio Personal de Mateo Balciunas

¡Bienvenido a mi portfolio personal! Este proyecto es una muestra de mis capacidades como desarrollador web. Construido con React y diseñado para ser moderno, interactivo y responsive. Aqui podrás explorar mis proyectos, conocer mis habilidades y contactarme.

## Características Destacadas

*   **Navegación Intuitiva**: Un menú de navegación fijo con iconos que permite un acceso rápido y cómodo a cada sección.
*   **Diseño Moderno y Minimalista**: Estética limpia con una paleta de colores hueso y azul eléctrico que muestra un contraste vibrante.
*   **Fondo Dinámico**: Un sutil patrón de puntos y esferas animadas en el fondo que añade un toque de dinamismo sin distraer del contenido principal.
*   **Secciones Claras y Completas:
    *   **Hero**: Una bienvenida clara.
    *   **About Me**: Información sobre mi trayectoria y pasiones.
    *   **Skills**: Una breve descripción de mis competencias técnicas.
    *   **Projects**: Muestra de algunos trabajos, presentados en un formato de lista horizontal.
    *   **Contact**: Un formulario de contacto funcional para facilitar la comunicación.
*   **Experiencia de Usuario Fluida**: Scroll suave entre secciones y un diseño responsive que se adapta a diferentes pantallas.
*   **Componentizacion con React**: Estrucutra modular y reutilizable para facilitar el mantenimiento y posibles modificaciones a futuro.


## Tecnologías Utilizadas

*   **Frontend**:
    *   [React](https://react.dev/)

## Instalación y uso local

Para tener una copia local de este proyecto funcionando, has de seguir estos pasos:
1.  **Clonar el repositorio**:
    ```bash
    git clone <https://github.com/mato-balciunas/my-portfolio.git>
    ````
2.  **Navegar al directorio del proyecto**:
    ```bash
    cd my-portfolio
    ````
3.  **Instalar dependencias**:
    ```bash
    npm install
    # o si usas yarn
    yarn install
    ````
4.  **Iniciar el servidor**:
    ```bash
    npm run dev
    # o si usas yarn
    yarn dev
    ````
